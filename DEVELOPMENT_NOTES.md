# Endpoints to Create

| Method | Endpoint | Desc                                |
|-|-|-------------------------------------|
| POST | /api/files/initiate | Initiate file upload and get fileId | 
| POST | /api/files/upload | Create a file                       | 
| GET | /api/files | Get all files                       |
| GET | /api/files/download/{fileId} | Download a file                     |
